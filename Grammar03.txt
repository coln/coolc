Grammar
=======
S -> A
A -> B + A | C * A
B -> int | (B) | (A)
C -> int | (C) | (A)

Bad Grammar


First/Follow
============
First(S) = { int, ( }
First(A) = { int, ( }
First(B) = { int, ( }
Follow(S) = { $ }
Follow(A) = { $, ) }
Follow(B) = { $, ), +, * }
Follow("int") = { $, ), +, * }
Follow("(") = { int, ( }
Follow(")") = { $, ), +, * }
Follow("+") = { int, ( }
Follow("*") = { int, ( }
